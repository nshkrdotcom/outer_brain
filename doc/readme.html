    OuterBrain

    OuterBrain is the semantic-runtime workspace above Citadel.

    It owns semantic journals, tool-manifest snapshots, prompt and strategy shaping, semantic quality checkpoints, live session fencing, and restart-safe reply publication.

    Scope

    • raw input normalization
    • semantic state and journaling
    • context assembly
    • prompt and strategy shaping
    • structured action-request synthesis
    • provisional and final reply publication
    • restart-safe downstream follow-up

    Status

    Active workspace buildout. The repo uses a non-umbrella workspace layout with core packages, bridges, a host surface, and proving examples.

    Development

    The project targets Elixir ~> 1.19 and Erlang/OTP 28. The pinned toolchain lives in .tool-versions.

    mix deps.get
mix ci

    The welded outer_brain_contracts artifact is tracked through the prepared bundle flow:

    mix release.prepare
mix release.track
mix release.archive

    mix release.track updates the orphan-backed projection/outer_brain_contracts branch so downstream repos can pin a real generated-source ref before any formal release boundary exists.
